# Package and release notes

## Extension

Current unpacked extension path:

```text
extensions/browser/
```

Chrome local install:

1. Open `chrome://extensions`.
2. Enable Developer Mode.
3. Load unpacked.
4. Select `extensions/browser/`.

## Library via jsDelivr

Current direct CDN URL:

```html
<script src="https://cdn.jsdelivr.net/gh/mcint/zipcodefirst-extension@main/library/zipfirst.js"></script>
```

For production, pin a tag instead of `main`:

```html
<script src="https://cdn.jsdelivr.net/gh/mcint/zipcodefirst-extension@v0.1.0/library/zipfirst.js"></script>
```
